# Calculator 📟

This is a simple calculator written in Python. It can add, subtract, multiply, and divide.

## How to Use ❔

1. Open calculator.py.
2. Input two numbers.
3. Choose operation: +, -, *, /
4. See the result.

## Example

Enter first number: 4 
Enter second number: 2  
Choose operation (+, -, *, /): *  
Result: 8
